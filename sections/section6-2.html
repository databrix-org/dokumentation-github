DHBW IT-Abteilung kontaktieren

Im letzten Schritt haben Sie Ihren Shibboleth-Service-Provider so konfiguriert, dass klar definiert ist, welche Daten von der IT-Abteilung benötigt werden. In diesem Abschnitt werden Sie die IT-Abteilung kontaktieren, damit Ihr Service-Provider bei ihnen registriert wird. Sobald Ihr Service-Provider registriert ist, können Sie jederzeit die Daten vom Nutzer erhalten.

Bitte erstellen Sie ein Ticket bei DHBW-ITS, um meinen Ship Owners Service Provider bei ihnen zu registrieren. Die folgenden Dateien werden benötigt. Bitte fügen Sie auch die untenstehenden Informationen in die E-Mail ein.

  • Metadaten-URL: https://example.org/Shibboleth.sso/Metadata (Bitte ersetzen Sie example.org durch Ihre Domain der Manager Node)
  • Gebrauchte Attribute: uid, mail, givenName, sn, etc.

Nachdem Ihre Anfrage bearbeitet wurde, können Sie die Daten vom Nutzer erhalten.
