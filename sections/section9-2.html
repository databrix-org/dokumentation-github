Persistent Volume

Für die Web-App benötigen wir ein persistent volume, um die Statische Styling, Python Files und Videos der Web-App zu speichern. Vorher im Schritt 6 haben wir schon ein Directory für JupyterHub erstellt. Wir können das Persistent Volume in gleichem Directory erstellen.

cd /opt/WebApp
mkdir static_volume
chmod 777 static_volume

Außerdem checken Sie noch einmal ob folgende Directory existiert. Hier werden die Media und python files gespeichert.

/opt/exchange_data/data_kurs_app
