NFS mounten

Im letzten Abschnitt haben Sie einen NFS Server auf Ihrem Manager Node eingerichtet. In diesem Abschnitt erfahren Sie, wie Sie dieses Network File System auf Ihren Worker Nodes mounten können. Damit die Worker Nodes die Dateien aus dem NFS-System lesen und schreiben können.

Worker Node

Die NFS Directory auf den Worker Nodes mounten: (Bitte die IP Adresse des Manager Nodes ersetzen)

cd /opt
sudo mkdir mount
sudo mount ipadresse_manager:/opt/exchange_data /opt/mount

Persistent User Volumen

Erstellen Sie ein Verzeichnis für die persistenten User Volumen:

mkdir jupyterhub-user-volumes
