Network File System

In diesem Abschnitt erfahren Sie, wie Sie einen NFS Server auf Ihrem Manager Node einrichten können. Solange ein Container für die Programmierumgebung gestartet wird, wird dieses NFS-System automatisch mit diesem Container verbunden, sodass die benötigten Dateien immer im richtigen Container vorhanden sind. In diesem Abschnitt erfahren Sie, wie Sie dieses Network File System in Ihrem Swarm-Cluster konfigurieren können.

Manager Node

mkdir /opt/exchange_data
nano /etc/exports

Add the IPV4 Addresses of the all Worker Nodes to the file, separated by a space. Please replace the ip_worker1, ip_worker2, ip_worker3, etc. with the actual IP addresses of your Worker Nodes!


/opt/exchange_data ip_worker1(rw,no_subtree_check,no_root_squash,sync) ip_worker2(rw,no_subtree_check,no_root_squash,sync) ip_worker3(rw,no_subtree_check,no_root_squash,sync) ...

Then export the directory:


sudo exportfs -ra 
sudo service nfs-kernel-server restart

Worker Node

cd /opt
sudo mkdir mount
sudo mount ipadresse_manager:/opt/exchange_data /opt/mount
