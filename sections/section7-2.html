JupyterHub Datenbank

In diesem Abschnitt lernen Sie, wie Sie eine PostgreSQL Datenbank für JupyterHub einrichten. Wir benötigen diese Datenbank, um auf alle User Anfragen schnell zu antworten. Auch wenn die Webapp kurzzeitig ausfällt, bleiben die Daten erhalten.

Persistent Volume für Datenbank

Erstellen Sie ein Directory für die persistente Datenbank:

mkdir database-volume
